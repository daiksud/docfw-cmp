# 資料の充実度

**対応状況: 標準。** VitePress は公式ガイド、設定リファレンス、テーマ API を提供しています。このサイトでは、機能の可否を推測せず、標準機能とプラグインを分けて公式資料へ案内します。

| 調べたいこと | 公式資料 |
| --- | --- |
| Markdown の記法 | [Markdown Extensions](https://vitepress.dev/guide/markdown) |
| 画像と公開ファイル | [Asset Handling](https://vitepress.dev/guide/asset-handling) |
| ロケール、検索、テーマ | [Site Config](https://vitepress.dev/reference/site-config) |
| Vue の利用 | [Using Vue](https://vitepress.dev/guide/using-vue) |

::: info 実装固有の資料
Mermaid と自動サイドバーは公式の標準機能一覧だけでは判断できません。このサイトでは [Mermaid](/ja/criteria/mermaid) と [サイドバー生成](/ja/criteria/sidebar-generation) に、プラグインによる追加であることを明記しています。
:::

公式ドキュメントは更新されるため、バージョンに依存する仕様や設定を変更する前に、使用中の VitePress バージョンの資料と実際のビルド結果を確認してください。現在は v2 prerelease への移行期でもあるため、資料の広さをバージョン間の互換性の保証とは扱いません。リリース頻度の文脈は日付付きの[評価](/ja/assessment)に記録しています。

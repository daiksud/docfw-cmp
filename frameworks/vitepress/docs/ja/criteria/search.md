# ローカル検索

**対応状況: 標準。** このサイトは VitePress の `local` 検索プロバイダーを使っています。ビルド時に生成されるローカルの MiniSearch インデックスをブラウザーで検索するため、外部検索サービスは必要ありません。

右上の **検索** を開き、「Mermaid」「静的」「カウンター」などを入力してください。日本語ロケールでは、ボタン、結果なし、移動、閉じるなどの UI 文字列も日本語です。

::: tip 検索結果を良くするには
ページごとに明確な `title` と `description` を設定し、見出しと本文に読者が使う語を含めます。検索の分かち書きや一致の性質はインデックスとクエリに依存するため、専用の日本語形態素解析を追加した検索ではありません。
:::

検索はビルド済み文書を対象にします。認証済みの外部コンテンツや実行時 API を検索する仕組みではありません。

# 画像最適化

**対応状況: 部分対応。** `docs` 配下から参照するアセットは Vite のビルド処理を通り、出力名のハッシュ化や、設定されたサイズ上限内での data URL インライン化の対象になります。

```md
![同居アセット](./demo-asset.svg)
```

これはキャッシュに有利ですが、VitePress/Vite が画像を自動でリサイズしたり、形式変換したり、圧縮品質を最適化したりする機能ではありません。レスポンシブ画像や圧縮が必要な場合は、あらかじめ画像を生成する処理や専用の変換を追加します。

::: warning サイズの判断
インライン化の上限は Vite のアセット設定で決まります。画像の最適化方針を変えるなら、実際の出力サイズと配信要件を測定してから設定を調整してください。
:::

`public` 配下のアセットは別の扱いです。安定した URL が必要な場合は [アセット配置](/ja/criteria/asset-placement) を参照してください。

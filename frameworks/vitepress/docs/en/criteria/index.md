# Comparison Criteria

Each page below is both a concise finding and a runnable example. “Built-in”
means VitePress or its default theme supplies it in this site; “Plugin” means
this project installed and configured an integration; “Alternative” means a
different VitePress-native approach; and “Partial” names an important limit.

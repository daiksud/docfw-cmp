# Internationalization

**Built-in configuration:** VitePress supports
[localized sites](https://vitepress.dev/guide/i18n) through `locales`. This
site configures `en` and `ja` with separate labels, document text, and
local-search translations.

## Live locale navigation

Use the language navigation above to move between:

| Locale | Route | Content root |
| --- | --- | --- |
| English | [English documentation](/en/) | `docs/en/` |
| Japanese | [日本語ドキュメント](/ja/) | `docs/ja/` |

The routes are explicitly locale-prefixed, so links such as
`/en/criteria/gfm/` remain within English. The built-in locale switch keeps
the current relative path—for example, it links that page to
`/ja/criteria/gfm/`. VitePress does not translate or create the counterpart;
this project guarantees the switch by maintaining matching English and
Japanese page trees.

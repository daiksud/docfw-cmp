# Static Site Generation

**Evidence:** This sample emits prerendered static HTML at build time and then
hydrates for client-side transitions. The base text is present in the HTML,
which is not inherently an SEO disadvantage; enhanced search, navigation, and
components use JavaScript. No documented Nextra zero-JavaScript MPA mode was
found. The accurate description is static multi-route output with SPA-style
client transitions.

The static HTML is suitable for CDN hosting and gives visitors prebuilt
content. This page intentionally uses no external build-time data, keeping the
static build deterministic and available offline.

See the [assessment](/en/assessment) for the shared comparison with VitePress.

# MDX & MDC

MDX combines Markdown with JSX, so documentation can render React components alongside ordinary prose, lists, and code examples.

## MDC

MDC is the Markdown Components syntax used by Nuxt Content, including block component forms such as `::alert`. Nextra does not support MDC syntax. In Nextra, use MDX instead: import React components and render them as JSX, as shown with the counter and callout above.

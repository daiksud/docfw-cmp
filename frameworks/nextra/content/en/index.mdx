# Nextra

This site is the Nextra chapter of **docfw-cmp**, a Document Framework
Comparison project. It evaluates [Nextra](https://nextra.site/) — a
documentation framework built on Next.js — against a fixed set of criteria
applied consistently across several documentation frameworks, using this
running site as the evaluation artifact rather than a static report.

## About the comparison

The full list of frameworks and criteria is defined in the project's
comparison README:
[`frameworks/README.md`](https://github.com/daiksud/docfw-cmp/blob/main/frameworks/README.md).
Alongside Nextra, the project evaluates
[Fumadocs](https://www.fumadocs.dev/), [Docus](https://docus.dev/),
[Nuxt UI](https://docs-template.nuxt.dev/), [Starlight](https://starlight.astro.build/),
[Docusaurus](https://docusaurus.io/), and [VitePress](https://vitepress.dev/).

## This instance

The concrete stack used for this Nextra instance is Next.js 16 (App Router),
TypeScript, [Bun](https://bun.sh) as the package manager and task runner, and
[Pagefind](https://pagefind.app/) for local search.

## Where to go next

See the full criteria write-up → [`/en/criteria`](/en/criteria). It covers
GitHub Flavored Markdown compatibility, Markdown extensions, MDX/MDC,
Mermaid diagrams, sidebar generation, search, i18n, dark mode, image
optimization, static site generation, directory and asset placement,
repository complexity, CSS customization, and documentation richness — each
demonstrated on its own page rather than only described in prose.

For steps to install dependencies and run this site yourself, see
[`/en/running-locally`](/en/running-locally).

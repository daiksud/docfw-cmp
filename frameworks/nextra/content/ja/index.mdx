# Nextra

本サイトは、ドキュメントフレームワーク比較プロジェクト **docfw-cmp** における Nextra の
評価チャプターです。[Nextra](https://nextra.site/)（Next.js 上に構築されたドキュメント
フレームワーク）を、複数のドキュメントフレームワークに横断的に適用される一連の観点に
沿って評価します。静的なレポートではなく、この稼働中のサイト自体を評価対象の成果物と
しています。

## 比較プロジェクトについて

比較対象フレームワークおよび観点の一覧は、プロジェクトの比較用READMEである
[`frameworks/README.md`](https://github.com/daiksud/docfw-cmp/blob/main/frameworks/README.md)
に定義されています。Nextra のほか、[Fumadocs](https://www.fumadocs.dev/)、
[Docus](https://docus.dev/)、[Nuxt UI](https://docs-template.nuxt.dev/)、
[Starlight](https://starlight.astro.build/)、[Docusaurus](https://docusaurus.io/)、
[VitePress](https://vitepress.dev/) を評価しています。

## 本インスタンスについて

本 Nextra インスタンスで使用している具体的な技術スタックは、Next.js 16（App Router）、
TypeScript、パッケージマネージャ兼タスクランナーとしての [Bun](https://bun.sh)、そして
ローカル検索用の [Pagefind](https://pagefind.app/) です。

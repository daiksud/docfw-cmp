# Mermaid 図

Nextra はページレベルの追加設定なしに `mermaid` コードフェンスを図としてレンダリングします。

## フローチャート

```mermaid
flowchart TD
  A[Write MDX] --> B[Run Nextra]
  B --> C[Render page]
  C --> D[Read docs]
```

## シーケンス図

```mermaid
sequenceDiagram
  participant Reader
  participant Browser
  participant Nextra
  Reader->>Browser: Open docs page
  Browser->>Nextra: Request route
  Nextra-->>Browser: Rendered documentation
```

gantt
  title Documentation evaluation
  dateFormat  YYYY-MM-DD
  section Content

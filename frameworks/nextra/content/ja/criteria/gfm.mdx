# GFM互換性

Nextra は一般的な GitHub Flavored Markdown (GFM) 機能をサポートしています。Docs テーマでは、
マーカーの直後に空の引用行を置くことで、GFM アラートがテーマ付きコールアウトとして
レンダリングされます。

> [!NOTE]
>
> ノートは、内容をざっと読む場合でも役立つ補足情報に使います。

> [!TIP]
>
> ティップは、より良い・より簡単な方法についてのアドバイスです。

> [!IMPORTANT]
>
> インポータントは、読者が見逃すべきでない重要な情報を示します。

> [!WARNING]
>
> ワーニングは、混乱や予期しない挙動を招く可能性がある状況を説明します。

> [!CAUTION]
>
> コーションは、実行前に特に注意が必要な操作を示します。

- [x] GFMの例を書く
- [x] アラートの間隔を確認する
- [ ] 後でページを追加する

## インライン拡張

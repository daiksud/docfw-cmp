# Static Site Generation

**根拠:** このサンプルはビルド時に事前レンダリング済みの静的HTMLを出力し、その後
クライアントサイド遷移のために hydration します。本文はHTML内に存在するため、
それ自体がSEO上の不利になるわけではありません。一方、検索・ナビゲーション・
コンポーネントなどの拡張機能は JavaScript を使います。Nextra に文書化された
JavaScript なしの MPA モードは見つかりませんでした。正確には、SPA風の
クライアント遷移を備えた複数ルートの静的出力です。

静的HTMLはCDNホスティングに適し、訪問者にビルド済みのコンテンツを届けます。
このページでは外部のビルド時データを使用せず、静的ビルドを決定的かつオフラインでも
実行可能にしています。

VitePress との共通点は [評価](/ja/assessment) を参照してください。

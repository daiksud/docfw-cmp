# ドキュメントの充実度

Nextra の公式ドキュメント（https://nextra.site）は充実しており、ガイド・
ハウツー系のコンテンツ、Docsテーマ・Blogテーマ双方の完全なテーマ設定
リファレンス、ファイル規約（`_meta.ts`・`page.tsx`・コンテンツディレクトリの
配置など）、組み込みコンポーネントのリファレンスまでを網羅している。公式
サイト自体が Nextra で構築されている点も特筆に値する（ドッグフーディング）。
メンテナ自身が、利用者に提供しているのと同じレンダリングパイプライン・検索・
テーマを使っているということである。

ドキュメントの充実度・質は、本比較における他の観点（検索が機能するか、
ルートが404になるかなど、機械的に検証可能な項目）と比べて、本質的に主観が
入りやすい。読者には、本ページの所見だけでなく、公式ドキュメントを実際に
読んだ際の印象も併せて判断材料にしていただきたい。

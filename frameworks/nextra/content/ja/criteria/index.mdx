# 比較の観点

このセクションでは、docfw-cmp の各比較観点について、可能な限り実際に動作するデモと
ともに Nextra の対応状況をまとめています。

| 観点 | 対応 | ページ |
| --- | --- | --- |
| GFM互換性（Alerts） | ✅ | [GFM](/ja/criteria/gfm) |
| Markdown拡張 | ✅ | [Markdown拡張](/ja/criteria/markdown-extensions) |
| MDX, MDC | ✅ / ❌（MDCは非対応） | [MDX と MDC](/ja/criteria/mdx-mdc) |
| Mermaid | ✅ | [Mermaid](/ja/criteria/mermaid) |
| サイドバー自動生成 | ✅ | [サイドバー自動生成](/ja/criteria/sidebar-generation) |
| サイト内検索（ローカル） | ✅ | [検索](/ja/criteria/search) |
| 多言語対応（i18n） | ✅ | [多言語対応](/ja/criteria/i18n) |
| ダークモード | ✅ | [ダークモード](/ja/criteria/dark-mode) |
| 画像の自動最適化 | この Pages export では部分的 | [画像の自動最適化](/ja/criteria/image-optimization) |
| アセット配置 | ✅ | [アセット配置](/ja/criteria/asset-placement) |
| Static Site Generation | ✅ | [SSG](/ja/criteria/ssg) |
| `docs` ディレクトリの配置自由度 | ✅ | [docsディレクトリの配置](/ja/criteria/docs-directory) |
| リポジトリ・ディレクトリの複雑度 | VitePress より手数が多い | [リポジトリの複雑度](/ja/criteria/repo-complexity) |
| カスタマイズの柔軟性 | ✅ | [カスタマイズ](/ja/criteria/customization) |
| ドキュメントの充実度 | 基本は充実、深掘りは要調査 | [ドキュメントの充実度](/ja/criteria/documentation-richness) |

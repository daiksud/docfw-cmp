Documentation Framework Comparison

A side-by-side comparison of documentation-site frameworks — not just a feature checklist, but real, running example sites you can click through. Each example is built with a different framework and evaluated against the same fixed set of criteria.

Comparison Criteria

Every example site is evaluated against the same fixed set of criteria:

  1. GFM compatibility (especially GitHub Alerts)
  2. Markdown extensions
  3. MDX, MDC
  4. Mermaid support
  5. Automatic sidebar generation
  6. Site search (local search)
  7. Internationalization (i18n)
  8. Dark mode support
  9. Automatic image optimization
  10. Static Site Generation
  11. Freedom in docs directory placement
  12. Asset placement
  13. Repository / directory complexity
  14. Customization flexibility (especially styling / CSS)
  15. Documentation richness
